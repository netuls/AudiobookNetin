# 🐺 Stark CSAT

Sistema de registro de pontos de melhoria de CSAT do Squad Stark.

## Estrutura

```
stark-csat/
├── index.html        # Página principal
├── css/
│   └── style.css     # Estilos
├── js/
│   └── app.js        # Lógica da aplicação
├── assets/
│   └── logo.png      # Logo do Squad Stark (adicione aqui)
└── README.md
```

## Como usar

1. Faça o download ou clone este repositório
2. Coloque a imagem do Squad Stark em `assets/logo.png`
3. Abra o `index.html` no navegador — ou publique via GitHub Pages

## GitHub Pages

1. Vá em **Settings → Pages**
2. Em **Source**, selecione a branch `main` e a pasta `/ (root)`
3. Clique em **Save**
4. Acesse em: `https://seuusuario.github.io/stark-csat`
